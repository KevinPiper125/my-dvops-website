Pig Dice Game

A turn-based Android dice game — simple concept, solid execution.

← Back to Home

Overview

The Game

Pig is a classic turn-based dice game where each player rolls a die to accumulate points — but rolling a 1 wipes out your turn's score and passes the turn to your opponent. The strategy is knowing when to hold and bank your points versus pushing your luck for more.

I built this as a native Android app using Kotlin and Android Studio. The project was a great exercise in managing UI state, game logic, and turn-based interactivity on a mobile platform.

How It Works

Game Logic & UI

The app tracks each player's accumulated score, their current turn score, and the overall game state. Players can choose to roll — risking their turn total — or hold to bank their points. First to reach the target score wins.

Building this in Android meant wiring up button click listeners, updating the UI in real time as game state changed, and handling edge cases like win conditions and turn resets cleanly. It's a small app, but it covers a lot of Android fundamentals: Activities, Views, event handling, and state management.

Kotlin Android Studio Android SDK UI State Management Event Listeners

What I Learned

Key Takeaways

This project was about getting comfortable with the Android development lifecycle. Managing state across turns, updating the display in response to user actions, and thinking about what happens when things go wrong (what if a player closes the app mid-game?) are all real challenges even in a simple project like this.

It also helped me get faster in Android Studio — setting up emulators, debugging on device, and reading Logcat output efficiently.
